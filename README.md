# INSTALL DEPENDENCIES

```
    composer install
```

# GENERATE ENVIROMENT VARIABLES FILE

```
    cp .env.example .env
```

# GENERATE KEY

```
    php artisan key:generate
```

# CREATE DATABASE STRUCTURE

```
    php artisan migrate
```

# POPULATE DATABASE

```
    php artisan db:seed
```
